Scikit-rt

Scikit-rt is a Python toolkit for loading, plotting, and analysing radiotherapy data in DICOM and NIfTI formats. It includes image registration via elastix and NiftyReg, single- and multi-atlas segmentation, and region-of-interest (ROI) comparisons.

Documentation: https://scikit-rt.github.io/scikit-rt/
Code repository: https://github.com/scikit-rt/scikit-rt/
PyPI project page: https://pypi.org/project/scikit-rt/

Development supported by Cancer Research UK RadNet Cambridge [C17918/A28870].
